Planet Money Music
Planet Money is a podcast/blog/NPR program about the economy. The table to the left is an up to date list of all the episodes and music played in them as listed in NPR's API. Check out my Github repo to grab the code to get this list for yourself. If you just want to download the table in CSV format, click the download button at the top left corner of the table. Currently working on making a nicely formatted CSV file, stay tuned!

My (very) long-term goal for this data is to make a pie chart with genre breakdowns.
